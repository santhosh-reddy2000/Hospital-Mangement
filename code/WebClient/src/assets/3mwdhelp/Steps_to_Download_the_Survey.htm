Steps to Download the Survey

1.Log in to the MaxSurvey web portal

2.Navigate to the Well Details tab in the navigation pane,Click the well in the well list displayed of which the survey/surveys has to be downloaded

3.Click Corrected Surveys, Select the Run from the drop down,to download the surveys of the respective Run else select All, If you want to   download the corrected surveys of all the runs at once

—The portal displays the list of the corrected surveys available in the selected Run as shown

 

4.Click Download Corrected Surveys

—The portal downloads a comma separated text file (.CSV) as Corrected surveys containing the displayed survey information as shown.
