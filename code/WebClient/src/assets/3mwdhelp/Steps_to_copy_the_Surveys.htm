Steps to copy the Surveys

1.Log in to the MaxSurvey web portal

2.Select the surveys to be copied in the survey list, either from

—Raw Surveys

—Corrected Surveys

3.Select the Run number from the drop down, to copy the surveys of that specific run else, select All, to copy the surveys of all the runs available in the selected Well at once

4.Click Copy

—The web portal copies the survey information to the clipboard, in the text format

5.Displays a message prompt “Data Copied!”

6.Paste the copied information to a text file or, a spreadsheet

—Displays the information as shown.
