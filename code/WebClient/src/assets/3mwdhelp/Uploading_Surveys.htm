Uploading Surveys

Steps to Upload the Survey

1.Save the survey data in an Excel spreadsheet, using the appropriate template for the well

2.Log in to the MaxSurvey web portal

3.Click Well details in the navigation pane

—Click the Well from the display list, to which the survey data has to be uploaded

4.Click Upload File

5.Navigate to the folder where the survey spreadsheet file is saved, and select the spreadsheet

6.Click Open

7.Uploads the survey to the MaxSurvey server.


Surveys with Errors, cannot be Imported
Validation Errors in the survey are earmarked in Red
Discrepancies in data are earmarked in Yellow as a warning


Invalid surveys cannot be uploaded
